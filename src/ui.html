        How it works
        
        
        
          Exports variables from all local collections into
          DESIGN.md — an open, agent-agnostic design system file.
        
        
        
        
          Works with Cursor, Claude, Windsurf, Copilot, and any tool that reads project context. Pair with AGENTS.md for build rules.
        
        
        
        
          Enable Figma styles to include text style hyper tokens, paint colors, and shadows.
        
        
        
        
          Enable W3C DTCG when you need a standards-based tokens.json for Style Dictionary or other tooling.
        
        
        
        
          Use the Session tab to visit library files one by one, capture components, then export from your product file with merged lookup tables.
